Instalacion
Debido a no encontrar un webhosting gratuito para publicar el proyecto se concluyo a realizar este manual.
----------------------------------------------------------------------------------
REQUESITOS.

PYTHON 3.7 (por recomendacion al instalar seleccione el path) o configure en el entorno de variable
--------------------------------------------------------------------------------
librerias de ..
flask
chatterbot
chatterbot-corpus
---------------------------------------------------------------------------------
para instalar estas librerias mediante el CMD, 
pip install chatterbot-corpus 
pip install chatterbot
pip install flask 
-----------------------------------------------------------------------------------
luego estando en la ubicacion del archivo app.py
ejecutamos python app.py para poder correr como servidor la pagina.
-----------------------------------------------------------------------------------
en nuestro navegador
escribimos
http://localhost:5000/
